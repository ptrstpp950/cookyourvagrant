Install cookbooks from supermarket

There are a lot of cookbooks to use on https://supermarket.getchef.com/. To install download it or run:

cd vagrantwindows
mkdir cookbooks
git init
git add .
git commit -m "Init commit: change name and add network"
mkdir cookbooks
knife cookbook site install git -o ./cookbooks -b
knife cookbook site install windows -o ./cookbooks -b
knife cookbook site install iis -o ./cookbooks -b
knife cookbook site install chef_handler -o ./cookbooks -b
knife cookbook create mywebsite -o ./cookbooks
