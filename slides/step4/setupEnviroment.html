Setup environment

  • In cookVagrant folder create vagrantWindows
  • Add box with: 
     vagrant box add windows2012 file:///c:/vagrantfiles/windows_2012_r2_virtualbox.box
  • Run vagrant init and change box name to windows2012
  • Form https://supermarket.getchef.com/cookbooks/iis download IIS cookbook and unpack it in vagrantwindows\cookbooks
  • Same with https://supermarket.getchef.com/cookbooks/windows and https://supermarket.getchef.com/cookbooks/git
  • Create cookbook with 
    knife cookbook create mywebsite -o $PWD/cookbooks
