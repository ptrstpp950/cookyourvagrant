Build your own box

It is easy but time consuming. The basic build is simple

vagrant package --base [VM name from Virtual Box] --output /myfolder/myboxname.box

There are some "tools" like packer.io to automatize building your box for example with: https://github.com/joefitzgerald/packer-windows, but it takes hours to build a Windows box (download ISO, install updates, ngen .NET framework, etc.), but it is like: fire and forget :)
