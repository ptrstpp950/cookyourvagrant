To "see" result of our work we need to use one more vagrant feature: port forwarding. So under the VM name add following line 

config.vm.network "forwarded_port", guest: 8080, host: 9876
Also we need to reconfigure or stop iptables service. I will just turn it off with: 
config.vm.provision :shell, :inline  => '/etc/init.d/iptables stop'
Now we can run 
vagrant reload --provision
to reload machine and apply provisioning
