Configure box

Using box called base can be problematic, so let's add a new one with 

vagrant box add "centos-6.4" file:///[FULL PATH TO DIRECTORY]/CentOS-6.4-i386-v20131103.box
We can list already installed boxes with 
vagrant box list
And remove unused onces with (they consume space under $HOME/.vagrant.d/): 
vagrant box remove [BOX NAME]
