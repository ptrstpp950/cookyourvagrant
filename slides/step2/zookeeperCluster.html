Zookeeper cluster

To configure zookeeper cluster we have to set nodes in zoo.cfg and create myid with unique identifier of node

Section of zoo.cfg, which specifies all nodes look like bellow: 

# specify all zookeeper servers
# The fist port is used by followers to connect to the leader
# The second one is used for leader election
server.1=192.168.10.11:2888:3888
server.2=192.168.10.12:2888:3888
server.3=192.168.10.13:2888:3888
To achieve above we can write in ruby following: 
<% x = 1  %>
<% @nodes.each do |hostname| %>
server.<%= x %>=<%= hostname %>:2888:3888
  <%x += 1%>
<% end %>
