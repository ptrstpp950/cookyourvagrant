Run zookeeper

We can reuse zookeeper file which we copied to init.d in shell provisioning. File we can copy or download from here TODO URL, save it in templates/default and add in recipe template directive:


template "/etc/init.d/zookeeper" do
	source "zookeeper.initd"
	owner "root"
	group "root"
	mode "0755"
end

service "zookeeper" do
	supports :status => true, :restart => true, :reload => true
	action [ :enable, :restart ]
end
