Stop iptables

We already created" and "strated" a "service in chef. Now we just need to "stop" it. The new file is iptables.rb under recipes directory, with following content:


service "iptables" do
	supports :status => true, :restart => true, :reload => true
	action [ :disable, :stop ]
end

Now add invoke of this recipe in under both machines in vagrantfile with: 

chef.add_recipe "storm::iptables"

After vagrant provision everything should be OK:)
