Storm & supervisor- common parts

We need to install: storm, supervisor, so in the new file stormCommon.rb under recipes code is easy:

package "supervisor" do
	action :install
end

package "storm" do
	action :install
end

template "/etc/storm/conf/storm.yaml" do
	source "storm.yaml.erb"
	owner "storm"
	group "storm"
	mode "0644"
end

Content of template file templates/default/storm.yaml.erb is very simple:

nimbus.host: "192.168.33.10"
