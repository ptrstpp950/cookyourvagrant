Setup nimbus

We just need to copy file for supervisor, which can be copied form our previous provisioning or here: into templates/default directory

The nimbus.rb recipe looks like following

include_recipe "storm::stormCommon"

template "/etc/supervisord.d/supervisord-storm-nimbus.ini" do
	source "supervisord-storm-nimbus.ini"
	owner "root"
	group "root"
	mode "0644"
end

service "supervisord" do
	supports :status => true, :restart => true, :reload => true
	action [ :enable, :restart ]
end
