Basic commands and features

  1. To access to created machine, we can run now: 
    vagrant ssh
  2. If you receive error about missing ssh command just do prerequisites or use putty instead
  3. The default user and password for box is vagrant and vagrant, but public boxes have also ssh key authentication turned on
  4. After successful login run following code inside VM
    cd /vagrant
echo "shared" > echo.txt
  5. Now in host machine in current directory (with Vagrantfile) we can run
    cat echo.txt
    and see "shared" on screen.
  6. By default vagrant directory is "mounted"/"shared" with VM so it is easy to transfer files between
