Basic commands and features

  1. To access created machine, we now can run : 
    vagrant ssh
  2. If you receive error about missing ssh - do prerequisites
  3. The default user and password for box is vagrant and vagrant, but public boxes have also ssh key authentication
  4. After successful login run following code inside VM
    cd /vagrant
echo "shared" > echo.txt
  5. Now on the host machine in current directory (with Vagrantfile) we can run
    cat echo.txt
    and see "shared" on screen.
  6. By default vagrant directory is "mounted"/"shared" with VM so it is easy to transfer files between
