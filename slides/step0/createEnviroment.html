Let the battle begins

Create environment

  1. Download CentOS 6.4 i386 Minimal box from http://developer.nrel.gov/downloads/vagrant-boxes/CentOS-6.4-i386-v20131103.box
  2. Create dir (let's call it `storm-shell`) enter it and run: 
    
vagrant box add "base" file:///[FULL PATH TO DIRECTORY]/CentOS-6.4-i386-v20131103.box
vagrant init
vagrant up
